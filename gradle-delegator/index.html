What we learned?

Gradle uses the Kotlin compiler’s scripting engine. The generated class (commonly named Build_gradle) is the concrete implementation of your build script.

More about Build_gradle

build.gradle.kts is compiled to a class (e.g., Build_gradle).

Build_gradle extends KotlinBuildScript, which is tailored for Kotlin DSL support.

KotlinBuildScript uses ProjectDelegate to delegate calls to the actual Project (from org.gradle.api.Project).

This design gives you a type-safe DSL that directly maps to Gradle’s build system.
